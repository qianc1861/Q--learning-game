# Q--learning-game

A simple Q-learning grid-world game example (Python).

### Features

- **Environment**: A grid world (now 10x10 in the code) with start `S`, goal `G`, and several traps `X`.
- **Agent**: Learns, via Q-learning, to move from `S` to `G` while avoiding traps.
- **Demo**: After training, the agent automatically runs a trajectory using the learned policy, either printed in the terminal or visualized with Pygame.

### Environment setup

1. Install dependencies (a virtual environment is recommended):

```bash
pip install -r requirements.txt
```

2. Make sure you are using Python 3.10+.

### Run the game

Run in the project root:

```bash
python q_learning_game.py
```

The program will ask you for the number of training episodes (recommended 1000–3000, default 2000). During training it periodically prints the total reward for that episode, and after training it will show a trajectory generated by the learned policy.

You can follow the prompts to train and demo again.
